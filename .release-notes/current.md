## 修改点

- 破坏式重构 APK 课程运行引擎，课程业务数据只包含教材、章节、课程页、内容块和交互场景。
- 将课程内容改为严格类型化解析与渲染，解析后的数据直接进入 APK 内置场景，不再经过旧枚举、字符串参数或兼容分发层。
- 重构 APK 课程更新清单解析，只接受教材 ID、完整包和文件级下载地址、大小及 SHA-256 校验信息。
- 课程托管与 APK 发布彻底解耦；APK 默认不配置课程源，只有显式注入课程清单地址时才启用云端下载和更新检查。
- 本次 APK 发布只涉及架构和底层能力；真实课程数据、课程生成工具与课程发布渠道保持不变。

## 修复点

- APK 在启用课程前严格校验未知字段、ID、页码、内容块、场景参数、声明式图元、文件集合、PDF 文件头和页数；任何错误都会拒绝新包并保留旧缓存。
- 删除旧课程模型、旧课程协议、静默降级和空白可视化路径，避免错误课程被部分加载。
- 未配置课程源时不再显示无效的首次课程下载提示，也不会发起旧 GitHub 课程地址请求。
