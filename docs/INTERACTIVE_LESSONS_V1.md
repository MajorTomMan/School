# 交互课程 V1

本版本先验证两个高质量样板课，而不是批量生成模板课程。

## 一次函数

- 根据教材课程标题自动识别。
- 显示 `y = kx + b`、教材移动端摘要、来源页码和原页入口。
- 支持调节 `k` 与 `b`，输入真实 `x` 值。
- 函数图像、数据点、斜率三角形和计算结果同步更新。

## 牛顿第一定律

- 根据物理教材课程标题自动识别。
- 展示伽利略理想实验的思想过程、背景知识和常见误区。
- 支持调节初速度和摩擦系数，播放、暂停和重置运动。
- 运动画面、速度箭头、摩擦力、位置数据和速度—时间图同步更新。

## 内容边界

V1 的教材正文是依据教材内容整理的移动端摘要，不冒充逐字原文。绑定原 PDF 后可从课程直接跳到相应页核对。后续版本再接入公式、文章、插图和表格的结构化提取与移动端重排。
