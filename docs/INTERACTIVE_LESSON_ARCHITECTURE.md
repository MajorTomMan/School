# 交互课程架构

交互课程通过 `InteractiveLessonCatalog` 按学科和课程标题匹配专属页面。未匹配课程继续使用现有生成课程页面，不影响原有课程路径。

第一版使用 Jetpack Compose Canvas 实现实时绘图和物理模拟，目的是先验证课程体验、生命周期与性能。后续需要复杂公式排版、矢量演出或通用几何构造时，可以在保持同一课程规格接口的前提下接入 KaTeX、Rive 或 JSXGraph。

## 性能策略

- 只有牛顿实验处于播放状态时才逐帧计算。
- 暂停、离开页面或实验结束后停止帧循环。
- 图像根据参数直接重绘，不生成视频文件，也不把整套动画预加载到内存。
- 所有实验均可离线运行。
