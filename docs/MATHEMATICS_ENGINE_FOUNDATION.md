# School 数学深化内核

## 范围

数学深化层建立在精确表达式和单位量纲层之上，目标是覆盖初中到高中教材反复出现的数据结构与确定性步骤：

- 多项式
- 一次与二次方程
- 一次不等式
- 判别式
- 函数与定义域
- 二维、三维向量和几何对象
- 几何变换
- 证明步骤依赖

它不是通用计算机代数系统，也不自动证明任意数学定理。

## 多项式

`Polynomial` 使用“指数 -> 精确有理系数”的稀疏表示，支持：

- 加、减、乘
- 系数缩放
- 精确求值
- 多项式导数
- 除以线性因式 `(x-r)`
- 教材式文本渲染

例如：

```text
(x-2)(x-3) = x²-5x+6
```

全部系数都使用 `BigRational`，不会因为小数误差导致根判断漂移。

## 方程与不等式

`AlgebraSolver` 为一次和二次方程返回：

- 精确解表达式
- 近似值
- 结构化步骤
- 每一步理由
- 必要条件

二次方程先计算判别式，并根据课程声明的数域决定：

- 实数域无解
- 一个重根
- 两个实根
- 两个复根

`InequalitySolver` 首版处理一次不等式，明确记录“除以负数时反转不等号”。

## 函数和定义域

`FunctionDefinition` 绑定变量、表达式和定义域约束。首批约束包括：

- 分母不为零
- 偶次根号内非负
- 对数式参数为正

函数采样返回可缺失点，绘图层可以据此表现断点，而不是把不合法值强行连线。

## 几何

二维模型：

- `Point2`
- `Vector2`
- `Line2`
- `Circle2`
- `Transform2`

三维模型：

- `Point3`
- `Vector3`
- `Plane3`

支持距离、点积、叉积、夹角、向量投影、直线相交、平行、垂直、点到线距离、点到平面有符号距离和平面投影。

几何变换使用仿射矩阵，首版支持平移、旋转、缩放、关于 y 轴对称和变换组合。

## 证明步骤

`ProofStep` 区分：

- 已知
- 定义
- 定理
- 推理
- 等价变形
- 结论

每一步可以声明依赖的事实和新产生的事实。`ProofStructureValidator` 检查：

- 引用了尚未得到的事实
- 重复定义事实 ID

它只验证步骤依赖完整性，不声称已经证明陈述本身在数学上必然正确。定理适用条件将在具体教材章节中配置。

## 教材权限

课程必须显式允许多项式、方程、不等式、判别式、向量、几何变换或证明步骤。底层支持二次方程，不代表一次函数章节可以提前展示求根公式。
